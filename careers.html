Data Scientist (Closed)

Job Description

자사 서비스에서 생성되는 사용자 데이터를 분석하여 인사이트 도출 및 전략 수립을 지원합니다. 추천 시스템, 이상 탐지, 머신러닝 기반 모델 개발 등의 프로젝트를 주도합니다.

Requirements

  • 데이터베이스 운영 및 관리, 데이터 모델링 역량
  • Python, SQL 등 데이터 분석 언어에 능숙한 분
  • Pandas, Scikit-Learn 등 데이터 분석 라이브러리 활용 경험
  • 기초 통계 활용 능력과 기계학습 모델 개발 및 평가 경험

Preferred Qualifications

  • 정규화 기법, 인덱싱 전략 기반의 성능 최적화 능력
  • A/B Test를 통한 고객 데이터 사용성 평가 능력
  • KPI(Key Performance Indicators) 설계 역량

Application Developer (Closed)

Job Description

당사는 자사 서비스를 위한 모바일 애플리케이션 개발하고 있습니다. 사용자의 편의성과 성능을 고려한 애플리케이션 개발 및 유지보수를 함께할 개발자를 찾고 있습니다.

Requirements

  • 디자인 패턴의 이해도와 유지보수 역량
  • Kotlin, Swift, Java 등 애플리케이션 개발 언어에 능숙한 분
  • RESTful API 통신 및 데이터베이스 활용 경험
  • Git 등 버전관리 시스템 활용 역량

Preferred Qualifications

  • 자료구조, 알고리즘, 운영체제의 이해
  • 비동기 처리(프로세스 스케쥴링, 쓰레드, 코루틴)의 이해와 적용 역량
  • 네트워크 및 엔드포인트 보안 조치 구현 역량

Quantitative Analyst (Closed)

Job Description

당사 퀀트 애널리스트는 금융 데이터 분석 및 수학적 모델링을 통해 리스크 평가, 금융 상품 정량 분석을 수행합니다.

Requirements

  • 추론 통계 기반의 리스크 추정 및 포트폴리오 최적화
  • 거시경제 지표기반의 통계적 분석 역량과 성과 분석 지표의 이해
  • 금융시장 및 상품에 대한 이해 및 관련 프로젝트 경험
  • 회귀분석, 분산분석, 시계열분석 기반의 통계적 추정 역량

Preferred Qualifications

  • 기업 가치평가 방법론을 이해하고 적용가능한 분
  • Stochastic Volatility 기반의 확률적 수리모델링 역량

AI Engineer (Closed)

Job Description

당사 AI 엔지니어는 정형 데이터를 기반으로 머신러닝/딥러닝 모델을 개발하고 실제 서비스에 적용하는 업무를 수행합니다. 엔지니어링 역량과 함께 AI 모델의 기획, 개발, 운영을 고려한 MLOps를 구축하는 프로젝트에 참여하게 됩니다.

Requirements

  • MLOps와 데이터 파이프라이닝의 이해
  • Python, SQL 기반의 인공신경망 모델 개발 및 평가 경험
  • Linux 환경에서의 프로그래밍 경험
  • Scikit-Learn, Pytorch, Tensorflow 등 기계학습 라이브러리 활용 경험

Preferred Qualifications

  • 데이터와 모델의 병렬처리, 분산학습 역량
  • Transformer Architecture 기반의 신경망 모델링 역량
  • Deep Q-Learning 기반의 강화학습 역량
