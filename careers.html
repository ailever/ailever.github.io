Data Scientist (Closed)

Job Description

고객에게 전달할 서비스 데이터와 사용자 데이터를 분석하여 인사이트 도출 및 전략 수립을 지원합니다. 추천 시스템, 이상 탐지, 기계학습 기반 모델 개발 등의 프로젝트를 주도하며 서비스 데이터의 질적 향상을 도모합니다. 또한, 기업 정보시스템과 데이터 거버넌스 체제를 구축하며 서비스 안정성을 확보하기 위한 프로젝트에 참여하게 됩니다.

Key Responsibilities

  • 데이터 수집 및 정제
  • 데이터 정합성 및 품질 검사
  • 탐색적 데이터 분석(Exploratory Data Analysis, EDA) 수행
  • 확증적 데이터 분석(Confirmatory Data Analysis, CDA) 수행
  • 비즈니스 문제를 해결하기 위한 기계학습 기반 모델링
  • 데이터 시각화 및 커뮤니케이션

Requirements

  • 데이터베이스 운영 및 관리, 데이터 모델링 역량
  • Python, SQL 등 데이터 분석 언어에 능숙한 분
  • Pandas, Scikit-Learn 등 데이터 분석 라이브러리 활용 경험
  • 기초 통계 활용 능력과 기계학습 모델 개발 및 평가 경험

Preferred Qualifications

  • 정규화 기법, 인덱싱 전략 기반의 성능 최적화 능력
  • T-Test, A/B Test, ANOVA를 통한 고객 데이터 사용성 평가 능력
  • KPI(Key Performance Indicators) 설계 역량

Application Developer (Closed)

Job Description

당사는 자사 서비스를 위한 모바일 애플리케이션 개발하고 있습니다. 사용자의 편의성과 성능을 고려한 애플리케이션 개발 및 유지보수를 함께할 개발자를 찾고 있습니다.

Key Responsibilities

  • 고객 요구사항 분석 및 반영
  • 애플리케이션 설계, 개발, 유지보수(CI/CD)
  • 애플리케이션 단위 테스트와 디버깅
  • 개발 문서화와 결과 보고서 작성

Requirements

  • 디자인 패턴의 이해도와 유지보수 역량
  • Kotlin, Swift, Java 등 애플리케이션 개발 언어에 능숙한 분
  • RESTful API 통신 및 데이터베이스 활용 경험
  • Git 등 버전관리 시스템 활용 역량

Preferred Qualifications

  • 자료구조, 알고리즘, 운영체제의 체계적 이해
  • 비동기 프로그래밍(Scheduling, Multithreading, Coroutine) 역량
  • 네트워크 및 엔드포인트 보안 조치 구현 역량

Quantitative Analyst (Closed)

Job Description

당사 퀀트 애널리스트는 고객에게 전달할 유의미한 정보를 도출하기 위해, 금융 데이터 분석 및 수학적 모델링을 통해 리스크 평가, 금융 상품 정량 분석을 수행합니다.

Key Responsibilities

  • 금융 모델 개발 및 구현
  • 금융 데이터 분석 및 처리
  • 위험 관리 및 시나리오 설계 및 분석
  • 금융 시장 조사와 연구 논문 분석

Requirements

  • 추론 통계 기반의 리스크 추정 및 포트폴리오 최적화
  • 거시경제 지표기반의 통계적 분석 역량과 성과 분석 지표의 이해
  • 금융시장 및 상품에 대한 이해 및 관련 프로젝트 경험
  • 회귀분석, 분산분석, 시계열분석 기반의 통계적 추정 역량

Preferred Qualifications

  • 기업 가치평가 방법론을 이해하고 적용가능한 분
  • 파생상품, 자산운용, 금융공학 모델 이해도
  • Stochastic Process 기반의 수리모델링 역량

AI Engineer (Closed)

Job Description

당사 AI 엔지니어는 정형 데이터를 기반으로 머신러닝/딥러닝 모델을 개발하고 실제 서비스에 적용하는 업무를 수행합니다. 엔지니어링 역량과 함께 AI 모델의 기획, 개발, 운영을 고려한 자사 MLOps를 구축하는 프로젝트에 참여하게 됩니다.

Key Responsibilities

  • 데이터 수집과 전처리
  • 도메인 최적화 모델 설계
  • 학습 시나리오 구성 및 최적화 파라미터 도출
  • 모델 배포 및 운영
  • 지속적인 성능 모니터링 및 개선

Requirements

  • MLOps와 데이터 파이프라이닝의 이해
  • Python, SQL 기반의 인공신경망 모델 개발 및 평가 경험
  • Linux 환경에서의 프로그래밍 경험
  • Scikit-Learn, Pytorch, Tensorflow 등 기계학습 라이브러리 활용 경험

Preferred Qualifications

  • 데이터와 모델의 병렬처리, 분산학습 역량
  • Transformer Architecture 기반의 신경망 모델링 역량
  • Deep Q-Learning 기반의 강화학습 역량
